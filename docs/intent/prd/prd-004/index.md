# PRD-004 Multidimensional Documentation Governance

This Body Package defines a multidimensional [[DH-HYGIENE-PROFILE#documentation-hygiene-profile|hygiene profile]]
built from three [[DH-HYGIENE-MATURITY|maturity levels]],
independent [[DH-CAPABILITY-DIMENSION|capability dimensions]], and explicit
[[DH-RULE-ACTIVATION|rule execution states]]. It also unifies semantic reference
and dependency behavior through typed [[DH-GOVERNANCE-EDGE|governance edges]].

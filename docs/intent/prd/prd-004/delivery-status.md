# Delivery Status

In progress. SPEC-003 now defines cumulative profile evaluation, N/A and
suppression semantics, the legacy maturity mapping, edge normalization, and
evidence required for each delivery slice. The implementation now centralizes
compatibility-family metadata and registers atomic invariants. The versioned
`profile` report combines project facts, independent execution decisions,
dimensional targets and observations, N/A rationale, and invariant evidence.
Legacy maturity mapping, conflict detection, suppression non-evidence, and an
optional below-target CI gate are executable and covered by fixtures.

Subsequent slices normalize semantic references, pinned references, derivations,
and projections into one ordered edge record and expose deterministic graph,
Fan, and cycle metrics. Explicit thresholds run through the independent
`governance.topology` family. The controlled structure slice now adds a reusable
document-template registry, deterministic merge, binding coverage, stable
diagnostics, profile evidence, and inline-profile compatibility. The governed
structure slice adds revision compatibility windows, exact profile pins,
read-only migration planning, and atomic compatible migration. This repository
binds all five profiles to one revisioned shared template; its structure target
is governed and all configured targets pass. The dependency selector slice now
parses heading selectors, resolves deterministic canonical ATX slugs, preserves
localized signatures, records selector evidence on normalized edges, and emits
`DH_SELECTOR_001`. The reference-model slice now routes Wiki Link, Markdown
Link, and frontmatter declarations through a versioned occurrence IR; explicit
syntax-and-context policy feeds one syntax-independent edge normalizer while
preserving diagnostics and edge JSON. The scoped-anchor slice adds frontmatter
multi-anchor declarations, file/block SHA-256 verification, compatible file
JSON, and default-off local commit verification with per-item evidence.
Deterministic reverse transitive impact now propagates every resolved semantic
edge kind into the graph report and dependency profile evidence. The identity
lifecycle slice now validates status obligations and
`supersededBy` across assets and package identities, rejects terminal targets,
and reports ordered authority migrations. Budgets, trends, and audited exception
execution remain undelivered.

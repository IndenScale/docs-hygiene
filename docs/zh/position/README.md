## 变更路由

- 中文共享产品含义的表示进入 `docs/zh/engineering/ul/` 下的对应术语文件，并与 canonical UL 保持身份和结构一致。
- 期望产品行为的变化进入 `docs/zh/engineering/prd/` 下的原子成员。
- 变更范围内的验收与证据进入 Issue。
- 对解释模型的调整进入本目录。
- 对已交付行为的主张必须回到根目录代码、配置和测试核验。

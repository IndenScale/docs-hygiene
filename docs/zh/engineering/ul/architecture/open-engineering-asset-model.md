# 开放工程资产模型

开放工程模型只固化两类文档结构，并把另外两类交付角色保持为位置无关：

- UL 保存共享语言与长期约束；
- PRD 保存产品意图、边界与需求；
- Issue 承载变更范围内的验收、协作与交付证据；
- Artifact 指任何实现或证据对象，包括代码、测试、配置、生成物、SDK 内容与提交。

只有 UL 与 PRD 要求仓库文档结构。Issue 与 Artifact 由适配器和关系识别，而非固定目录。
模型不再规定 Definition、Implementation、Glossary、SDK 或精化层级轴。

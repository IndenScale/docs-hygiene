# C-021 确定性图社区

社区发现只消费 C-020 已解析的类型化边。算法把不同的有向端点对转换为无权无向图，使用
确定性深度优先遍历寻找桥，删除桥后把剩余连通分量报告为社区。社区 ID 为
`community:<字典序最小成员>`。算法没有随机种子；重复引用和并行 relation 不改变成员。

关系图报告有序社区成员及跨社区的不同有向边。可选 `communityBaseline` 把身份映射到
期望社区 ID，并为变化、缺失或新发现成员生成有序变化记录。默认只发现和报告，不阻断。

只有 `enforceCommunityBaseline: true` 才把基线变化提升为 `DH_TOPOLOGY_006` 错误。
没有该显式开关的基线不能阻断 CI；Fan、循环、预算和超级节点策略保持独立。

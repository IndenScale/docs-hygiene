# C-014 传递影响

每条规范化语义边都从依赖方 source 指向权威 target。一个受管身份变化时，沿这些边的
反方向可达的所有其他受管身份都受到影响，包括间接依赖方。语义引用、锁定引用、派生和
投影复用同一传播算法。

关系图报告以 `transitiveImpact` 公开有序映射：key 是变化身份，value 是排序、去重后的
受影响身份列表。空影响集合不输出；变化身份自身被排除；visited-set 遍历确定性地终止
循环。重复边和并行边不会重复计数，未解析端点不传播影响。

该分析只报告证据，不夹带策略：它不推断哪些文件已变化，不排序严重度，不执行影响预算，
也不通知负责人。预算与例外执行仍属于独立的拓扑和工作流能力。

# 验证

Fixture 必须证明注册表顺序确定、规则归属完整、成熟度累积、目标与检测结果分别报告、
N/A 排除、disabled 与 suppression 不构成证据、旧值映射、确定性模板合并、完整模板绑定
覆盖、内联 Profile 兼容、兼容与阻断场景下的原子模板迁移、边兼容、标题 selector 解析与
本地化签名同位、带版本的引用 occurrence 收集、显式非语义策略、语法无关规范化、
file 与 block 变化隔离、逐项多锚诊断、opt-in commit 校验、不同邻居 Fan 指标、
去重的反向传递闭包、确定性循环终止、确定性循环组、状态义务、叶子与资产权威迁移、
本地化后继同位、终止目标拒绝、可选阈值执行，以及稳定的组合文本与 JSON 画像输出。

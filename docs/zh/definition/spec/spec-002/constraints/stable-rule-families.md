# C-002 稳定规则族

公共规则 ID 是 `project.entry-docs`、`docs.structure`、`documents.contracts`、
`localization.parity`、`concepts.references`、`governance.identity`、
`governance.traceability` 和 `adapters.external`。
SPEC-003 随后在这些 ID 之后增加 `governance.topology`，不改变原有名称和相对顺序。

这些 ID 分别治理项目入口文档、文档结构、文档契约、语言同位、概念引用、治理身份、
相邻层级追溯和外部工具编排；新增规则治理显式拓扑阈值。规则名称与输出顺序属于公共
契约。

# Definition Layer

本目录保存 Definition Layer。它的 Reference Library 是 Glossary，受治理 Body 是
Spec 与 Test Definition。

## 权威边界

- Glossary 把 UL 含义投影为稳定的 Definition 身份。
- Spec 使用这些身份形式化来源 PRD。
- Test Case、模型、Oracle 和 Verifier 是 Definition Body。
- Test Result 和运行观察属于独立 Evidence 平面。

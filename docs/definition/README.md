# Definition Layer

This directory contains the Definition Layer. Its Reference Library is the
Glossary and its governed Bodies are Specs and Test Definitions.

## Authority

- The Glossary projects UL meaning into stable Definition identities.
- A Spec formalizes a source PRD using those identities.
- Test cases, models, oracles, and verifiers are Definition Bodies.
- Test results and runtime observations belong to the separate Evidence plane.
